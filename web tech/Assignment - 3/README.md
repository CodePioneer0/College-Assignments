# Faculty Profiles - CST Department, IIEST

This HTML page displays faculty member profiles for the Computer Science & Technology Department at Indian Institute of Engineering Science and Technology (IIEST).

## Features

1. **Search Functionality**: Users can search for faculty members by name
2. **Complete Profile Information**: Each profile includes:
   - Full name with proper title (Dr./Mr./Ms.)
   - Professional photograph
   - Email contact information
   - Phone numbers
   - Highest qualification
   - Current semester subjects
   - Research areas

## How to Use

1. Open `faculty-profiles.html` in a web browser
2. Use the search box to find specific faculty members
3. Click "Show All" to display all profiles
4. Add faculty photographs to the `images/` folder with the correct filenames

## Image Requirements

- Place faculty photographs in the `images/` folder
- Use appropriate filenames as referenced in the HTML
- Recommended image size: 200px width × 250px height
- Supported formats: JPG, PNG
